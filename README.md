# ElShreif
 El Shreif Academy Academy System For Manage Student, Teaceher, Student Attendance, Student Salary, Teacher Salary and Teacher invoice.
 
#Requirements 
- XAMPP, LAMP or WAMP
- PHP 8.1.1
- Composer version 2.2.9
- must install (imagick) To View Student Barcode
